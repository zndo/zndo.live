Spring Boot-1

发表于 2017-06-14 | 分类于 Spring Boot |

简介

至 2017-8-1，Spring Boot 当前 GA 版本为 1.5.6.RELEASE。

  • 项目主页
  • 参考指南
  • API
    Spring Boot 使创建可“即时运行”的独立的，产品级别的 Spring 应用变得很容易。Spring Boot 会选择最合适的 Spring 子项目和第三方库进行整合，大多数 Spring Boot 应用只需要极少的 Spring 配置。

功能特性

  • 创建独立的 Spring 应用
  • 直接内置 Tomcat，Jetty 或者 Undertow（不需要部署 WAR 文件，非强制）
  • 提供特定的 ‘starter’（启动器） POM 文件来简化你的 Maven 配置
  • 尽可能的自动配置 Spring
  • 提供生产环境就绪的功能，例如 metrics-指标，health checks-健康检查，和 externalized configuration 外部配置
  • 绝对没有代码生成且不需要 XML 配置
    参考指南包含所有功能特性的详细描述和常见的使用案例。
